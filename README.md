# Backend API

Backend API for the SportsOS project built using Node.js, Express.js, and MongoDB. This layer handles client requests, business logic, database operations, and returns API responses.

## Technologies Used

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication

## Folder Structure

* **controllers/** – Handles API requests and responses.
* **models/** – MongoDB schemas.
* **repositories/** – Database operations.
* **middleware/** – Authentication and request middleware.
* **config/** – Database configuration.
* **utils/** – Helper functions.
* **index.js** – Application entry point.

## Features

* Authentication APIs
* Academy APIs
* Athlete APIs
* Coach APIs
* Sports APIs
* Reviews APIs
* Enquiry APIs
* Shortlist APIs
* Admin APIs

## How to Run

1. Install dependencies:

```bash
npm install
```

2. Create a `.env` file with the required environment variables.

3. Start the server:

```bash
npm start
```

or

```bash
node index.js
```
